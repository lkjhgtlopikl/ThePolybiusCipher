## Квадрат Полибия

Программа кодирует русский или английский текст (язык выбирается с помощью radiobutton) методом квадрата Полибия.
Закодированный текст может состоять из цифр или символов русского/английского алфавита, что также выбирается с помощью
radiobutton

## ГОСТ 28147-89

ГОСТ 28147-89 — устаревший государственный стандарт СССР (а позже межгосударственный стандарт СНГ), описывающий алгоритм
симметричного блочного шифрования и режимы его работы. Полное название — «Системы обработки информации. Защита
криптографическая. Алгоритм криптографического преобразования».

Принцип работы алгоритма основан на сети Фейстеля:

1. Генерируется ключ длиной 64 байта и разбивается на 16 подключей длиной 4 байта для 16 раундов обработки
2. Генерируется прямой и обратный S-боксы для кодирования и декодирования блока текста
3. Исходный текст разбивается на блоки по 8 байт
4. Если текстовый блок короче 8 байт он дополняется пустыми байтами в конце
5. Текущий блок разбивается на левый и правый подблоки
6. Далее начианются 16 раундов обработки блоков
   1. Выполняется сложение левого блока по модулю 2 с подключем текущего раунда
   2. Выполняется кодирование левого блока с помощью S-бокса
   3. Выполняктся сложение по модулю 2 с правой частью
   4. Правая и левая части меняются местами (кроме последнего раунда)
7. Декодирование текста происходит в обратном порядке
   1. Правая и левая части меняются местами (кроме последнего раунда)
   2. Выполняктся сложение по модулю 2 с правой частью
   3. Выполняется кодирование левого блока с помощью S-бокса
   4. Выполняется сложение левого блока по модулю 2 с подключем текущего раунда
