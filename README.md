## Квадрат Полибия
Программа кодирует русский или английский текст (язык выбирается с помощью radiobutton) методом квадрата Полибия.
Закодированный текст может состоять из цифр или символов русского/английского алфавита, что также выбирается с помощью radiobutton

---
## Метод одиночной перестановки
Программа кодирует произвольный текст методом одиночной перестановки.
